ABOUT THE PROJECT

Heritage Treasures

Objective

Analyze global distribution, risk status and regional inscription trends of 1,121 UNESCO World Heritage Sites from 1978–2019.

Core Analysis

Country/state concentration, sites in danger, regional trends and exact-coordinate geographic distribution.

Technology

CSV, Tableau, Tableau Public, Python Flask, HTML and CSS.

Project Flow

Dataset → Preparation → Tableau → Dashboard → Tableau Public → Flask → User.
